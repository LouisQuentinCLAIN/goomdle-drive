# GOOMDLE DRIVE

## About 

+ This project is a simple online drive focused on pictures. 

## Dependencies

+ node 15.0.1 && npm 7.0.3
+ vue-cli / vue.js

## Install 

Install vue-cli
```
$ npm install vue@cli --global

```
Install packages
```
cd goomdle-drive
npm install
```
Run
```
$ npm {your-parent-directory}/goomdle-drive start
```

Access via [browser](http://localhost:3000)
